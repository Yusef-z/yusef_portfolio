Signly - (Graduation Project)

Signly is a mobile app that eases the communication barrier between signers and non-signers by providing a translation tool that coverts text to sign and vice versa. and also has a dictionary which contains detailed translation of any given work in multiple sign language systems.
the data used in this application was scraped from the website "spreadthesign.com" and then uploaded to firebase firestore so that it can be used in flutter

Movie Decider - (Still Under Development)

Movie Decider is an application that can help users decide what movie to watch based on their preferred genre and year range by returning a random movie that matches their preferences.
also the application will show similar movies in case the returned movie doesn't intreset the user.
also users can keep track of their watched movies/shows so that they won't appear in the search.
the application relies on the TMDB api to get the data
